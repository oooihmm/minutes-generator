### 📌 개요

- **구현 기간**: 2023.03.09 ~ 2023.03.24
- **기술 스택**: Python

> 회의 영상 파일을 텍스트 회의록으로 변환·요약하는 자동화 애플리케이션

### 🎯 목적

연구실에서는 행정 업무 등을 위해 회의 후 내용을 다시 확인하고 정리해야 하는 경우가 발생합니다.

그러나 20~30분 내외의 회의 영상을 직접 다시 시청하며 핵심 내용을 정리하고 요약하는 과정은 생각보다 많은 시간이 소요됩니다.

본 프로젝트는 이러한 반복적인 정리 작업에서 발생하는 시간적 부담을 줄이고 회의 내용을 간편하게 문서 형태로 확보할 수 없을까라는 고민에서 출발하였습니다.

연구실 내에서 주요 메신저로 Slack을 사용하고 있다는 점을 고려하여,
별도의 서비스 이동 없이 Slack 환경 안에서 바로 사용할 수 있는 봇으로 구현하였습니다.

### ⚙️ 구조

본 애플리케이션은 회의 영상 파일을 입력으로 받아,
다음과 같은 처리 과정을 거쳐 회의록을 생성합니다.

1. Slack에서 업로드된 회의 영상 파일을 수신 (Socket Mode)

2. 영상 파일에서 음성 데이터를 추출하여 텍스트로 변환 (Whisper)

3. 변환된 텍스트를 기반으로 회의 내용 요약 (GPT + PAPAGO)

4. 생성된 회의록을 Slack 스레드를 통해 사용자에게 전달

각 단계는 순차적으로 수행되며, 실패 시 이후 단계가 실행되지 않도록 처리했습니다.
